Мои проекты

За моими плечами несколько завершенных проектов разного масштаба и направленности: от игр и десктоп-приложений до веб-сервисов. Каждый из них чему-то меня научил: работе в команде, архитектурному мышлению, умению доводить до результата

Сейчас я в процессе создания нового проекта, который станет следующим шагом в моем развитии

Завершенные проекты

Super Rabbit Platformer | Pygame

2D-платформер в стиле классического Марио: уровни, сбор монет, враги, физика прыжков.

Роль и вклад: Парный проект. Отвечала за физику движения персонажа, коллизии, анимации и генерацию уровней. Училась писать игровую логику без переусложнения, декомпозировать задачи и синхронизировать код через Git в паре

Стек: Python, Pygame, Git

Чему научилась: - ООП в игровом контексте (классы Player, Enemy, Level) - Работа с событиями, отрисовкой кадров и FPS - Первый опыт конфликтов в Git и их разрешения

Трекер для диабетиков | Qt

Десктоп-приложение для учета уровня сахара, инъекций инсулина и приема пищи. Помогает визуализировать динамику и не пропускать замеры.

Роль и вклад: Индивидуальный проект. Полный цикл: проектирование интерфейса в Qt Designer, написание логики на PyQt5, работа с SQLite для хранения записей

Стек: Python, PyQt5, SQLite

Чему научилась: - Строить desktop-приложения с графическим интерфейсом - Проектировать БД и связывать её с интерфейсом - Думать о пользователе: делала интерфейс интуитивным, добавляла подсказки и цветовые маркеры

Веб-сервис для отслеживания привычек | Django

Полноценный веб-сайт, где пользователи могут создавать привычки, отмечать выполнение, получать статистику и напоминания на почту.

Роль и вклад: Командный проект, 5 человек. Моя зона ответственности - разработка бэкенда: модели, распределение задач между участниками и

Работали по Git-флоу, делали код-ревью, вели таск-трекер. Я также участвовала в обсуждении архитектуры БД и помогла команде с настройкой окружения.

Стек: Python, Django, DRF, Celery, Redis, PostgreSQL, HTML/CSS (базово).

Чему научилась: - Командная разработка - это про коммуникацию и ответственность - Писать чище и читаемее, чтобы коллегам было легко вникать - Попробовала себя в роли team lead-а

Текущий проект

Игра про последний день Романовых | Unity

Сюжетная 2D-игра, посвященная трагическим событиям 1918 года. В разработке: атмосфера, диалоги, исследования локаций, принятие решений.

Роль и вклад: Парный проект. Моя задача - гейм-дизайн, написание сценария, часть логики на C# и работа с ассетами.

Стек: Unity, C#, Git

Что осваиваю сейчас: - Unity-разработку с нуля - Совмещение кода и нарратива в игровом движке - Работу с анимациями и тайлами
