### Super Rabbit Platformer | Pygame

*2D-платформер в стиле классического Марио: уровни, сбор монет, враги, физика прыжков.*

**Роль и вклад:**
Парный проект. Отвечала за **физику движения персонажа, коллизии, анимации и генерацию уровней**. Училась писать игровую логику без переусложнения, декомпозировать задачи и синхронизировать код через Git в паре

**Стек:** Python, Pygame, Git

**Чему научилась:**
-  ООП в игровом контексте (классы Player, Enemy, Level)
-  Работа с событиями, отрисовкой кадров и FPS
-  Первый опыт конфликтов в Git и их разрешения

*Десктоп-приложение для учета уровня сахара, инъекций инсулина и приема пищи. Помогает визуализировать динамику и не пропускать замеры.*

**Роль и вклад:**
**Индивидуальный проект.** Полный цикл: проектирование интерфейса в Qt Designer, написание логики на PyQt5, работа с SQLite для хранения записей

**Стек:** Python, PyQt5, SQLite

**Чему научилась:**
-  Строить desktop-приложения с графическим интерфейсом
-  Проектировать БД и связывать её с интерфейсом
-  Думать о пользователе: делала интерфейс интуитивным, добавляла подсказки и цветовые маркеры
