# Почему я выбрала Material?

Я выбрала тему Material из MkDocs, потому что она сочетает в себе лаконичный дизайн и высокую функциональность. Тема автоматически подстраивается под разные устройства, т.е. сайт будет одинаково хорошо выглядеть и на компьютере, и на планшете, и на телефоне, что важно для пользователей и потенциальных работодателей. В теме Material интуитивно понятный интерфейс, что не мало важно для новых посетителей сайта. А встроенная поддержка русского языка и переключение между светлой и тёмной темами создают комфортные условия для чтения. Кроме того, тема обеспечивает качественную подсветку синтаксиса для кода, что особенно важно для лабораторных работ по программированию.
